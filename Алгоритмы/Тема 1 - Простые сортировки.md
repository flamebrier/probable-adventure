1.Реализовать сортировку выбором.

**Входные данные.**
В первой строке задано одно целое число n (1 ≤ n ≤ 1000) — количество элементов в массиве. Во второй строке задано n целых чисел через пробел: a0, a1, …, an−1 — элементы массива.

**Выходные данные.**
Выведите n чисел через пробел, где число i — это индекс минимального неотсортированного элемента на i-й фазе алгоритма.

**Примеры.**

1._Input_:

`5`

`40 30 20 40 20`

_Output_:

`2 4 4 3 4`

2._Input_:

`3`

`1 2 3`

_Output_:

`0 1 2`

2.Реализовать сортировку пузырьком.

**Входные данные.**
В первой строке задано одно целое число n (1 ≤ n ≤ 1000) — количество элементов в массиве. Во второй строке задано n целых чисел через пробел: a0, a1, …, an−1 — элементы массива.

**Выходные данные.**
Выведите целое число – количество инверсий (перестановок элементов в паре) в процессе сортировки методом «пузырька».

**Примеры.**

1._Input_:

`3`

`7 2 5`

_Output_:

`2`

2._Input_:

`10`

`12 7 92 5 18 4 32 48 11 74`

_Output_:

`18`
